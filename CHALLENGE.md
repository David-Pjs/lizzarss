# The Lizzarss Challenge

We built an AI that controls your computer with plain English. You press F9, type what you want, and your PC does it. Open apps. Fill forms. Type stuff. No clicks needed.

Now we are daring you to take it further.

**GitHub:** github.com/TeenovateX-Labs-Open-Source/lizzarss

---

## How to get started

1. Get a free Groq API key at [console.groq.com](https://console.groq.com) (1 minute, no credit card)
2. Install Python from [python.org](https://www.python.org/downloads/)
3. Fork the repo at [github.com/TeenovateX-Labs-Open-Source/lizzarss](https://github.com/TeenovateX-Labs-Open-Source/lizzarss)
4. Paste your Groq key into `agent.py` line 18
5. Run: `python agent.py`
6. Press F9 and see it work
7. Now build something nobody has built yet

## Team formation

Teams of 1 to 3 people.

You need at least one person who can write Python. Designers, testers, and idea people all count. One team, one GitHub fork.

---

## The three tracks

### Track 01: Most Useful

Build something that solves a real problem for a real person. Not a demo. Something that actually works and someone would actually use today.

### Track 02: Most Creative

The most surprising, unexpected, or strange use of Lizzarss. We do not know what this looks like. That is the point.

### Track 03: Hardest Technical

Record yourself doing something on your PC once. Lizzarss watches, learns the steps, and does it automatically forever after. Nobody has built this yet. First one to ship it wins.

## Rules

- Fork the repo and build on top of it
- Must run on a real Windows PC, no fake demos
- Submit a 2-minute screen recording of it actually working
- Code must be public on GitHub
- No AI-generated code you do not understand

---

## How to submit

Fork the repo. Build your thing. Record 2 minutes of it working on a real screen. Drop your GitHub link and video in the group.
